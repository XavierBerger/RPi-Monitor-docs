RPi-Monitor¶

RPi-Monitor is a software designed to monitor the metrics from GNU/Linux system and connected peripherals. It store statistic into local Round Robin Database and embed a web server allowing to display current status and statistics. This documentation is presenting the features provided by RPi-Monitor and explains how to install, configure and use it.
